### Базовые типы данных:
**int (integer)** - это целочисленный тип данных, который представляет любое целое число без дробной части. Например, 42 или -123.  

**str (string)** - это тип данных, который содержит последовательность символов, которые могут быть буквами, цифрами, знаками пунктуации и любыми другими символами. Строки в Python записываются в кавычках, например, "Hello, World!".  

**float** - это числовой тип данных, который представляет дробные числа. Например, 3.14 или -0.5.  

**bool (boolean)** - это логический тип данных, который может иметь только два значения: True (истина) или False (ложь). Булевы значения используются для выражения логических операций, таких как "и", "или" и "не".  

**list** - это тип данных, который содержит упорядоченный набор элементов различных типов данных. Списки обозначаются квадратными скобками и элементы разделяются запятыми. Например, [1, "two", 3.0, True].  

**set** - это тип данных, который содержит неупорядоченный набор уникальных элементов различных типов данных. Множества обозначаются фигурными скобками и элементы разделяются запятыми. Например, {1, 2, 3, 4}.  

**tuple** - это тип данных, который содержит упорядоченный набор элементов различных типов данных, похожий на список. Основное отличие состоит в том, что кортежи неизменяемы и обозначаются круглыми скобками. Например, (1, "two", 3.0, True).  

**dict (dictionary)** - это тип данных, который содержит неупорядоченный набор пар ключ-значение. Словари обозначаются фигурными скобками, где каждая пара ключ-значение записывается как "ключ: значение", разделенные запятой. Например, {"name": "Alice", "age": 25}.

### Методы списков list():
**append()** - добавление элемента в список reverse() - переворот списка index() - определение индекса элемента в списке  
**count()** - подсчет количества элементов в списке  
**del** - удаление элемента из списка  
**len(list)** - количество элементов  
**max(list)** - максимальный элемент  
**min(list)** - минимальный элемент  
**sum(list)** - сумма элементов  
**sorted(list, reverse=True - опционально)** - создание нового отсортированного списка  

**all(list)** - истина, если ВСЕ элементы True  
**any(list)** - истина, если ХОТЯ БЫ ОДИН элемент True  

*Пример: проверить, что все числа в паре четные*  
```pair = [4, 12]  
if all(x % 2 == 0 for x in pair):  
    print("Оба четные!")
```

### Методы строк str():  
**split()** - разделение строки на список строк  
**join()** - объединение строк в одну строку с указанным разделителем  
**replace()** - замена подстроки в строке  
**find()** - поиск первого вхождения, вернет индекс или -1  
**count()** - подсчет количества элементов в строке  
**strip()** - удаление начальных и конечных пробельных символов в строке  

**Конкатенация и умножение строк**: s = '1' **+** '0' **\*** 20  # Строка: единица и двадцать нулей  

### Методы множеств set():  
**add()** - добавление элемента в множество  
**count()** - подсчет количества элементов в множестве  

### Работа с файлами open():  
**open()** - открытие файла для чтения или записи  
**readline()** - чтение первой строки из файла  
**readlines()** - чтение всех строк из файла и возврат в виде списка строк  
**read()** - чтение всего содержимого файла в одну строку  

### Целочисленное деление и остаток (разбор числа на цифры)  
```
x = 123  
print(x // 10)  # 12 (отрезали последнюю цифру)   
print(x % 10)   # 3  (получили последнюю цифру)
```

### Перевод в 10-тичную систему из любой другой  
**int(строка, основание)**  
```
print(int('1011', 2))    # 11  
print(int('AF1', 16))    # 2801
```

### Перевод из 10-тичной системы (возвращают строки!)  
**bin(n)** - в 2-сс
**oct(n)** - в 8-сс
**hex(n)** - в 16-сс
```
n = 25  
print(bin(n)[2:])  # '11001' (срезаем 0b)   
print(oct(n)[2:])  # '31'    (срезаем 0o)  
print(hex(n)[2:])  # '19'    (срезаем 0x)
```
  
### УСЛОВИЕ IF  
```
if x % 2 == 0:  
    print("Число четное")  
elif x % 3 == 0:  
    print("Число делится на 3")  
else:  
    print("Число не подошло под условия")
```

### ЦИКЛ FOR  
**range(старт, стоп, шаг)** — стоп не включительно!  
```
for i in range(1, 10, 2):   
    print(i)  # Выведет 1, 3, 5, 7, 9
```

### ЦИКЛ WHILE  
```
n = 123  
while n > 0:  
    digit = n % 10  # Получили последнюю цифру  
    print(digit)
    n //= 10        # Отрезали её
```
  
**break** — немедленно выходит из цикла. Полезно, если ты нашел нужное значение и дальше искать нет смысла.  
**continue** — перепрыгивает к следующему шагу цикла, пропуская оставшийся код под ним.
